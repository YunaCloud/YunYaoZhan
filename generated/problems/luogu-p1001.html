洛谷 P1001 A+B Problem

这是一份示例题解，用来演示你的 ACM 站点会怎么展示 Markdown 内容。

题意

给定两个整数 a 和 b，输出它们的和。

思路

  • 这题没有算法难点，直接读入输出即可。
  • 如果后面你有更复杂的题，可以在这里写贪心、DP、图论、数据结构之类的思路。
  • 站点会把这一部分和右边的 cpp 代码放在同一页。

复杂度

  • 时间复杂度：O(1)
  • 空间复杂度：O(1)

易错点

  • 注意输入输出格式。
  • 如果题目数据范围较大，记得把 int 换成 long long。

C++ Solution

content/code/luogu-p1001.cpp 
#include <iostream>

int main() {
    int a, b;
    std::cin >> a >> b;
    std::cout << a + b << '\n';
    return 0;
}
