AtCoder Beginner Contest 044

C - Tak and Cards

Description

yunayu 有 $n$ 张卡片，第 $i$ 张卡片$(1 \le i \le n)$上写有一个数字 $x_i$。她现在想从这些卡片里选出若干张，使得所选卡片上的数字的平均值恰好为 $A$。请问她有多少种选法？

其中 $n, x_i, A$ 均为不超过50的正整数。

Solution

数据范围非常小，可以进行一个 $d$ 的 $p$。

但是注意一个点设计状态的时候不要用平均值而要用总和！不然会在转移过程中丧失很多可行解，因为本题必须在整数处转移咕。

我的状态设计为，考虑前 $i$ 张卡片，所取卡片数字总和为 $j$ ，且卡片数量为 $k$ 时的方法数。然后对于每张卡片取或不取转移即可。

时间复杂度为 $O(n^4)$。

Code

void solve() {
    cin>>n>>yu; for(int i=1; i<=n; i++) cin>>a[i];
    f[0][0][0]=1;
    for(int i=1; i<=n; i++) {
        for(int j=0; j<=50*50; j++) {
            for(int k=0; k<=i-1; k++) {
                f[i][j][k]+=f[i-1][j][k];
                f[i][j+a[i]][k+1]+=f[i-1][j][k];
            }
        }
    }
    int __=0;
    for(int i=1; i<=50*50; i++) {
        for(int j=0; j<=n; j++) {
            if(i==yu*j) __+=f[n][i][j];
        }
    }
    cout<<__<<'\n';
}

D - Digit Sum

Description

对于整数 $b$ ($b \ge 2$) 和 $n$ ($n \ge 1$)，定义函数 $f(b, n)$ 为：

若 $n < b$，$f(b, n) = n$ 若 $n \ge b$，$f(b, n) = f(b, \lfloor n/b \rfloor) + (n \bmod b)$

现给定两个整数 $n$ 和 $s$，请找到满足 $f(b, n) = s$ 的最小整数 $b$ ($b \ge 2$)。如果不存在这样的 $b$，则输出 $-1$。

其中 $1 \le n, s \le 10^{11}$。

yunayu 看不懂题目，于是请你帮她写。

Solution

首先首先，不要被题目描述吓到，写了这么一大段东西，其实翻译过来就是一句话：

$$f(b, n)\ 是\ n\ 在\ b\ 进制下的数位和。$$

然后考虑这样一件事情：

很自然吧！很美丽吧！

那么我们很快可以得出两个分支的结果：

  • 对于一个非负整数，其在任意进制下的数位和大小都不超过其自身数值大小。

现在考虑 $s \lt n$。emmm 对于每一个固定的进制来说判断其是否合法都是很容易的，可是数据范围这么大，枚举是不行的……同时这种合法性是单点的，二分也是不行的……要怎么考虑呢……

那么对于这种情况，我们常常考虑根号分治，而这题恰好就是！

  1. $s \gt n$ ：这根本不可能有解！
  2. $s = n$ ：我们要让 $n$ 完全出现在最低位上，那么显然最小需要 $n+1$ 进制。

根号分治

说到底，根号分治其实是一种优雅的暴力算法。对于一部分，使用一种策略，对于另一部分，使用另一种策略。然而两种策略要么是在根号范围内的暴力，要么是在根号复杂度下的算法，可以使得总复杂度为根号级别。

那么对于本题，如果进制 $b$ 存在，无非只有 2 种情况：

对于第一种情况，直接暴力即可。如果第一种情况没有找到合法解，进入第二种情况的查找，如果再找不到，就是不存在了。

那么现在问题转化为判断：

  1. $2 \le b \le \sqrt{n}$
  2. $\sqrt{n} \lt b \le n$

发动一下注意力！一个非常关键的注意是，当进制数超过 $\sqrt{n}$ 且不超过 $n$ 时，在该进制下其位数一定恰好为两位！

不妨 $n=p \cdot b+q$ ，由于 $b$ 大于 $\sqrt{n}$，那么高位 $p$ 一定小于 $\sqrt{n}$！我们只要枚举 $p$ 并检验合法解即可！

总时间复杂度 $O(10 \cdot \sqrt{n})$，其中常数来自枚举数位和。

  • 是否存在 $b$ 满足 $\sqrt{n} \lt b \le n$， 使得 $n$ 在 $b$ 进制下的数位和为 $s$ ？

Code

void solve() {
    cin>>n>>s;
    if(s>n) cout<<-1<<'\n';
    else if(s==n) cout<<n+1<<'\n';
    else {
        for(int i=2; i*i<=n; i++) {
            int sum=0, t=n;
            while(t) {
                sum+=t%i; t/=i;
            }
            if(sum==s) {
                cout<<i<<'\n'; return;
            }
        }
        int ret=-1;
        for(int i=1; i*i<n; i++) {
            int p=i, q=s-i; 
            if(q<0) break;
            if((n-q)%p==0&&p<(n-q)/p&&q<(n-q)/p) {
                ret=(n-q)/p;
            }
        }
        cout<<ret<<'\n';
    }
}
